## What I'm Building

**[SyncRoute](https://syncroute.vercel.app)** is a real-time ride-matching system built around geospatial precision — OSRM routing, MongoDB 2dsphere indexing, live WebSocket tracking, and OCR-based driver verification. The goal was to prove that a solo-built system could match production-grade reliability expectations.

**[Anantha](https://anantha.live)** is a conversational interface over 23 years of ARGO oceanographic data. It combines SQL, vector search, and multimodal LLMs so researchers can query 50GB+ of ocean datasets in plain language — with LSTM anomaly detection and geospatial forecasting built in.

**[LearnityX](https://learnityx.netlify.app)** is an AI-powered technical interview platform that reads your resume and adapts its questions to your actual stack — not a generic question bank, but context-aware evaluation built on Gemini.

## Research

**ISRO – NRSC** *(2025 – 2026)*  
Mapped 20+ years of tropospheric NO₂ trends across 25 Indo-Pacific countries using OMI and Sentinel-5P TROPOMI data. Built a DINEOF pipeline to reconstruct missing satellite observations — R² = 0.78. Manuscript targeting *Science of the Total Environment*.

**DLRL – DRDO** *(2025)*  
Designed an air-gapped RAG system for classified document retrieval. LangChain + ChromaDB + LLaMA 3.2, with CrossEncoder reranking and semantic chunking — sub-second retrieval, zero external data exposure.
